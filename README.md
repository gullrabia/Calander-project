#  Calendar Project (PHP)

A simple and dynamic **Calendar Web Application** built using **PHP, JavaScript, CSS, and MySQL**.  
This project allows users to view a calendar interface and manage date-based data using a backend connection.

---

##  Features

- Dynamic calendar interface
- PHP backend logic
- JavaScript-based interactions
- MySQL database connection
- Clean and simple UI
- Lightweight and easy to understand codebase

---

##  Technologies Used

- **PHP**
- **JavaScript**
- **CSS**
- **HTML**
- **MySQL**
